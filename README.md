# Run and deploy Luxcurrency locally

**Prerequisites:**  Node.js

1. Install dependencies:
``` bash
npm install
```
2. Run the app:
``` bash
npm run dev
```
## Initial Development

1st version only supports INR to USD or Vice-Versa
